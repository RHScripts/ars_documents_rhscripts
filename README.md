# ars_documents
A simple documents with item using metadata edited for use with quasar inventory

**Instalation**

> qs-inventory/shared/items.lua
```lua
    ['document'] = {
        ['name'] = 'document',
        ['label'] = 'Document',
        ['weight'] = 1,
        ['type'] = 'item',
        ['image'] = 'document.png',
        ['unique'] = false,
        ['useable'] = true,
        ['shouldClose'] = true,
        ['combinable'] = nil,
        ['description'] = 'A piece of paper containing important information'
    },
```
